# Monaco Editor 教程与实践

## Monaco Editor 介绍

微软项目 Monaco Workbench 后来变成了 VSCode， 而 Monaco Editor，是从这个项目中成长出来的一个 web 编辑器。 Monaco 基于浏览器，VSCode 是基于 electron。

## 在框架中使用

社区中已经有很多对 monaco editor 封装的组件。其中 react 和 vue 是最常用的两个框架。

react： [react-monaco-editor](https://github.com/react-monaco-editor/react-monaco-editor)

vue： [vue-monaco-editor](https://github.com/matt-oconnell/vue-monaco-editor)
